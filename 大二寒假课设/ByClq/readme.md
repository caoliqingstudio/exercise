##如果打不开图形界面文件ByClq.exe ，请将点击[这里](http://www.xitongcheng.com/jiaocheng/dnrj_article_17689.html)
部分电脑打不开，网上能查到的解决方案基本都用了，因为在部分电脑是可以使用的，如果不行请直接使用exe文件夹
中的main.exe文件，使用方法是在cmd下调用，参数为文件名（可以带路径），支持多个文件同时输入。
例如：如果您想处理 abc.c、def.c文件，那您应该输入 main.exe abc.c def.c/
      如果仅仅想处理一个，main.exe abc.c/即可
